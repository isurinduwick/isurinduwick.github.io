About Me

Mobile & Web Developer with Real-World Impact

I’m a Mobile & Web Developer with hands-on industry experience, specializing in building production-ready, scalable, and user-focused applications.

My core expertise lies in Flutter for cross-platform mobile app development (Android & iOS) and Laravel for backend and web systems. I’ve worked on real-world applications currently live in production, handling everything from UI/UX implementation to API integration, database management, and performance optimization.

I enjoy taking full ownership of projects, working independently, and collaborating closely with clients to transform ideas into reliable digital products. I’m recognized for my fast learning ability, strong logical thinking, and analytical problem-solving skills, which help me adapt quickly to new technologies and project requirements.

I’m also actively exploring AI integration in modern mobile and web architectures to build smarter and more efficient applications.
